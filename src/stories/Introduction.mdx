# Bem vindo ao Storybook do Valorant Tips V2

Aqui você encontrará os principais componentes base usados para construir as interfaces do Valorant Tips, e poderá usar os componentes testando as variações disponíveis

## Sobre

A versão 2 do Valorant Tips é na verdade a segunda grande versão desse projeto, inicialmente ele foi criado em react, posteriormente migrado para nextjs, e agora está passando por uma grande atualização, que envolve alteração no layout, um novo figma do zero, a api está sendo reescrita para usar clean archteture, e o frontend está sendo migrado para o novo nextjs e sendo recriado do zero. O novo front agora estará usando atomic design também
